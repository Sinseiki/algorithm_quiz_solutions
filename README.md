# Algorithmic Quizzes and Solutions

This repository demonstrates algorithmic problem solving
and cross-language implementation across TypeScript, Kotlin, Java, and Rust.

## What this repository demonstrates
- Multi-language implementations (TypeScript, Kotlin, Java, Rust)
- Understanding of algorithmic patterns and cross-language translation
- Problem descriptions are independently rewritten based on problems from Baekjoon Online Judge (BOJ)

## Featured Problems
- [Dijkstra (1504)](https://github.com/Sinseiki/algorithm_quiz_solutions/tree/main/04_Graph_Algorithms/1504_Shortest_Path_via_Two_Nodes)
  → Shortest path via mandatory nodes, implemented in 4 languages

- [LCS (9251)](https://github.com/Sinseiki/algorithm_quiz_solutions/tree/main/01_Dynamic_Programming/9251_LCS_Length_DP)
  → Dynamic programming with memory optimization
